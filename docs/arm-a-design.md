# Arm A "巧手" — 精密操作臂完整设计

## 1. 设计目标与约束

### 核心指标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 重复定位精度 | ±0.2mm | 超越 myCobot 280 的 ±0.5mm |
| 自由度 | 6-DOF | 全空间姿态操作 |
| 有效臂展 | ~350mm | 覆盖 300×300mm 工作台 |
| 额定负载 | 200g | 电子元件+末端执行器 |
| 自重 | <3kg（不含底座） | 桌面级 |
| 最大速度 | ~200mm/s | 非高速场景 |
| 控制频率 | ≥1kHz | 实时伺服环 |

### 设计原则

1. **精度优先**: J1\~J3 使用谐波减速器消除回差，这是超越 myCobot 的关键
2. **混合制造**: 金属骨架(刚性) + 3D 打印外壳(快速迭代) + CNC 关键件(精度)
3. **可手工装配**: 所有零件可用常规工具装配，不需要压力机或专用治具
4. **模块化关节**: 每个关节是独立模块，可单独测试和更换

## 2. 运动学构型

### DH 参数 (Denavit-Hartenberg)

```
        Z1 (上)                  
        │                       
        │  d1=80mm              
        │                       
  ──────●────── X1  J1: 底座旋转 (Yaw)
        │
        │
        ●──────────────────● J2→J3
        J2                 │ a2=200mm (上臂)
      (Shoulder)           │
                           ●──────────────● J3→J4
                           J3              │ a3=150mm (前臂)
                         (Elbow)           │
                                           ●─●─● J4/J5/J6 (腕部)
                                           d6=60mm
```

| 关节 | 类型 | θ 范围 | d (mm) | a (mm) | α (°) |
|------|------|--------|--------|--------|-------|
| J1 底座旋转 | 旋转 | ±170° | 80 | 0 | -90 |
| J2 肩部俯仰 | 旋转 | -30°~+150° | 0 | 200 | 0 |
| J3 肘部俯仰 | 旋转 | -150°~+30° | 0 | 150 | 0 |
| J4 腕部旋转 | 旋转 | ±180° | 0 | 0 | -90 |
| J5 腕部俯仰 | 旋转 | ±120° | 0 | 0 | 90 |

**最大臂展**: 200 + 150 + 60 = 410mm

## 5. 末端执行器 — 快换系统

### 5.1 快换接口设计

采用 **三球三槽运动学耦合 + 磁吸固定**，可在数秒内切换工具：

```
工具侧 (公端):              臂侧 (母端):
  ┌──────────┐              ┌──────────┐
  │ 3×钢球   │  ←对准→     │ 3×V槽    │
  │ Ø6mm     │              │ 120°分布  │
  │          │              │          │
  │ 圆形磁铁 │  ←吸合→     │ 圆形磁铁  │
  │ Ø15×3mm  │              │ Ø15×3mm  │
  │          │              │          │
  │ pogo pin │  ←电气→     │ pogo pad  │
  │ 4pin     │              │ 4pin     │
  └──────────┘              └──────────┘
```

- 3 颗 Ø6mm 钢球嵌入 3D 打印体，与对侧 3 条 V 槽配合 → **6 点约束，精确复位**
- Ø15×3mm 钕铁硼磁铁提供 ~5N 吸力 → 足够固定 200g 以下工具
- 4-pin pogo pin 连接器 → 传递电气信号 (真空泵控制、夹爪舵机 PWM)
- 全部可 3D 打印 + 淘宝采购标准件

## 6. 电气系统

### 6.1 系统框图

```
24V 电源 ──┬── 24V总线 ── TMC2209 #1 ── J1 NEMA17
           │              TMC2209 #2 ── J2 NEMA17
           │              TMC2209 #3 ── J3 NEMA17
           │              TMC2209 #4 ── J4 NEMA14
           │              TMC2209 #5 ── J5 NEMA14
           │              TMC2209 #6 ── J6 NEMA11
           │
           ├── 24V→5V (降压) ── STM32F407 主控板
           │                    ├── I2C ── AS5600 ×6 (编码器)
           │                    ├── UART ── TMC2209 ×6 (UART配置)
           │                    ├── STEP/DIR ── TMC2209 ×6 (脉冲控制)
           │                    ├── USB ── 上位机 (中间件)
           │                    ├── GPIO ── 限位开关 ×6
           │                    └── GPIO ── 急停按钮
           │
           └── 5V 分支 ── 真空泵 / 电磁阀 / 夹爪舵机
```

### 6.2 主控板方案

**推荐: STM32F407 最小系统板**（~30 CNY）

| 资源需求 | 数量 | STM32F407 可用 |
|---------|------|---------------|
| STEP 脉冲输出 | 6 路 | 定时器 PWM，充足 |
| DIR 方向输出 | 6 路 | GPIO，充足 |
| I2C (AS5600) | 1-2 总线 | I2C1 + I2C2 |
| UART (TMC2209) | 1-6 路 | UART1~6，充足 |
| GPIO (限位+急停) | 7+ | 充足 |
| USB | 1 | USB OTG |
| 主频 | 168MHz | 1kHz 控制环绰绰有余 |

**备选: ESP32-S3**（~35 CNY）— 自带 WiFi/BLE，但实时性略逊于 STM32。

### 6.3 TMC2209 步进驱动配置

| 参数 | J1/J2/J3 设置 | J4/J5/J6 设置 |
|------|-------------|-------------|
| 电流 (RMS) | 1.0-1.5A | 0.5-0.8A |
| 微步细分 | 256 | 256 |
| StealthChop | 启用 | 启用 |
| 接口 | UART 配置 + STEP/DIR 控制 | 同左 |

256 微步 + 50:1 谐波 = 每圈 200×256×50 = **2,560,000 步/圈**  
→ 角分辨率: 360°/2,560,000 ≈ **0.000141°/步**  
→ 在 350mm 臂展末端: ~0.00086mm/步 → **远超 ±0.2mm 需求**

### 6.4 接线规范

- 电机线: 4 芯屏蔽线, 0.5mm² (22AWG), 长度按需裁剪
- 编码器线: 4 芯 (VCC/GND/SDA/SCL), 0.2mm², 带屏蔽
- 所有线缆通过 3D 打印走线夹固定于臂杆表面
- 关节处预留弯曲余量, 使用硅胶软线
- 急停按钮: 常闭触点, 独立于主控, 直接切断电机驱动电源

### 装配顺序

```
阶段 1: 关节子组装 (并行)
├── [1A] J1 关节: 轴承压入底座 → 装谐波 → 装电机 → 装编码器
├── [1B] J2 关节: 轴承压入支架 → 装谐波 → 装电机 → 装编码器
├── [1C] J3 关节: 同 J2
├── [1D] J4 关节: 电机+行星减速组装 → 装入3D打印壳体
├── [1E] J5 关节: 同 J4
└── [1F] J6 关节: 同 J4, 加装快换母端

阶段 2: 臂杆组装
├── [2A] 上臂: 碳纤管 + 两端端堵 (结构胶固化 24h)
└── [2B] 前臂: 碳纤管 + 两端端堵 (结构胶固化 24h)

阶段 3: 总装 (必须顺序执行)
├── [3A] J1 装到底座外壳
├── [3B] J2 支架连接到 J1 输出法兰
├── [3C] 上臂插入 J2 支架, 紧定螺丝固定
├── [3D] J3 支架安装到上臂另一端
├── [3E] 前臂插入 J3 支架
├── [3F] J4→J5→J6 依次安装到前臂末端
└── [3G] 布线: 沿臂杆表面走线, 走线夹固定

阶段 4: 电气联调
├── [4A] 驱动板接线 + 电机测试 (逐关节, 无负载)
├── [4B] 编码器读数验证 (每关节正反转, 确认线性)
├── [4C] 限位开关测试
├── [4D] 急停功能测试 (最重要!)
└── [4E] 全关节协调运动测试

阶段 5: 标定
├── [5A] 零位标定: 各关节回零 + 限位开关触发位置记录
├── [5B] 运动学标定: 已知点测量, 修正 DH 参数
└── [5C] 精度测试: 重复定位 10 次, 计算标准差
```

### 关键注意事项

1. **轴承安装**: 交叉滚子轴承对安装面平面度要求高 (0.02mm)。如果 CNC 件到手后平面度不够，用细砂纸 (1000#) 在平板玻璃上研磨
2. **谐波减速器**: 安装前阅读厂家说明书，注意预紧力矩要求。不要过度拧紧
3. **碳纤管粘接**: 环氧胶需固化 24 小时。固化前用 V 型块确保管材与端堵同轴
4. **接线**: TMC2209 电机接线顺序错误会导致电机异响或不转，先用万用表确认线圈对
5. **螺纹胶**: 所有受振动的螺丝涂乐泰 243，但轴承和谐波安装螺丝**不要涂**（需要可拆卸）

## 11. 固件与标定 (概要)

### 固件架构

```
main.c
├── motion_control.c    // 1kHz 伺服环: 梯形加减速 + S曲线
├── kinematics.c        // 正/逆运动学 (基于 DH 参数)
├── encoder.c           // AS5600 I2C 读取, 多圈计数
├── tmc2209.c           // UART 配置 + STEP/DIR 脉冲发生
├── safety.c            // 限位检测, 急停处理, 软限位
├── comm.c              // USB CDC 通信, 接收/解析上位机指令
└── calibration.c       // 零位标定, 运动学参数存储 (Flash)
```

### 标定流程

1. **电气零位**: 各关节触发限位开关 → 记录编码器原点
2. **运动学标定**: 用卡尺测量末端到多个已知点的距离 → 最小二乘法修正 DH 参数
3. **重复精度测试**: 末端移动到同一目标点 20 次，用千分尺或游标卡尺测量偏差
4. **负载补偿**: 带载/空载下测量同一点位偏差 → 补偿重力影响

## 12. 风险与替代方案

| 风险 | 影响 | 替代方案 |
|------|------|---------|
| 谐波减速器价格过高 | 预算超标 | J1/J3 改用同步带 2 级减速 (~15:1×3:1=45:1), 精度退化到 ±0.4mm |
| CNC 加工周期长 | 项目延期 | 关节支架改为铝板线切割 + 折弯, 更快但刚性略差 |
| 碳纤管尺寸不标准 | 端堵配合不良 | 改用铝管 (6063-T5), 重量增加但尺寸更标准 |
| AS5600 精度不够 | 闭环效果差 | 升级为 AS5048A (14位) 或 AMT102 (可编程分辨率) |
| 步进电机丢步 | 运动失控 | 启用 TMC2209 StallGuard 堵转检测; 编码器闭环补偿 |
| 3D 打印件刚性不足 | 整体精度下降 | 外壳改为碳纤增强尼龙 (PA-CF), 或增加壁厚 |
