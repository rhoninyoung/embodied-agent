# Embodied Agent — 物理世界自主智能体

> 让 AI Agent 突破屏幕的边界，通过摄像头感知、机械臂操作、3D打印制造，  
> 在物理世界中实现 **感知 → 决策 → 执行 → 反馈** 的完整闭环。

## 愿景

构建一个由 AI 驱动的物理操作系统。AI 不再只是生成代码和文字，而是能够：

- **看见** — 通过摄像头观察工作台和零件
- **思考** — 理解任务、规划步骤、设计零件
- **制造** — 通过 3D 打印机生产结构件
- **操作** — 通过机械臂进行抓取、装配、布线
- **迭代** — 根据反馈优化设计，最终实现硬件自举

## 快速开始

> 详见 [路线图](docs/roadmap.md) 了解分阶段实施计划。

### 前置条件

- Python 3.10+
- USB 摄像头（推荐带 PTZ 云台）
- 桌面机械臂（如 Elephant Robotics myCobot 280 或类似产品）
- 拓竹 Bambu Lab P2S 3D 打印机
- 基本电子元件和工具

### Phase 0 — 环境搭建

1. 克隆本仓库
2. 按照 [硬件清单](docs/hardware-bom.md) 准备设备
3. 按照 [架构文档](docs/architecture.md) 配置软件环境
4. 开始与 AI Agent 协作！
