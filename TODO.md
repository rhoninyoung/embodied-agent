## 当前阶段: Arm A "巧手" 制造准备

### 阻塞关系

```
OpenSCAD 模型 (CNC 件)
  → 导出图纸 → 发给加工厂报价/生产
                                        ──→ 零件到齐 → 装配 → 标定 → 软件联调
OpenSCAD 模型 (3D 打印件)                    ↑
  → 切片 → 打印                              │
                                             │
淘宝采购 (电机/轴承/电子件) ─────────────────┘

STM32 固件 ──→ 电气联调 (需要硬件到齐)

中间件 WebSocket 服务 ──→ 上位机联调 (需要固件完成)
```
