3. **制定 72 小时生存计划**：基于地形，AI 规划：
   - 安全营地位置
   - 饮水方案
   - 第一座窑的位置（靠近粘土+燃料）

## Era 1: 电力求生（Day 1 - Day 30）

**核心目标**: 在笔记本彻底没电之前，制造出某种可持续电源为它充电。

### 方案 A：伏打电堆（最快，Day 1-3 可尝试）

```
最简易电池：铜 + 铁 + 酸性溶液

    ┌──────┐ ┌──────┐ ┌──────┐
    │ Cu   │ │ Cu   │ │ Cu   │    铜片：自然铜锤扁
    │ 果汁 │ │ 果汁 │ │ 果汁 │    酸液：发酵果汁 或 木灰水浸泡植物
    │ Fe   │ │ Fe   │ │ Fe   │    铁片：磁铁矿碎片 (导电性差) 或陨铁
    └──┬───┘ └──┬───┘ └──┬───┘
       串联      串联      串联
       ~~~~~~~~~~~~~~~~~~~~~~~~
       ~20 节 → 约 10-15V
```

**现实问题**：
- 每节仅 ~0.5V, 0.05-0.1A
- 20 节串联得到 ~10V，但电流极小（~50mA）
- 笔记本最低需要 ~15V 2A = 30W，伏打电堆远远不够正常运行
- **但可以**：拆开笔记本，直接给锂电池涓流充电（3.7V × ~3A = ~11W）
  - 需要 8 节铜铁电池串联 ≈ 4V
  - 以 50mA 充电，约 100-200 小时充满
  - 勉强可行但极慢，仅作应急备份

**可行性: ★★☆☆☆** — 应急续命用，不是长期方案

### 方案 B：水力发电机（Day 5-15，正解）

这是真正的长期解决方案。需要先完成一些前置工作。

**Day 1-3: 前置材料**

| 材料 | 获取方式 | 难度 |
|------|---------|------|
| 自然铜 | 河床淘洗，或在火成岩区寻找绿色/蓝色矿脉（孔雀石/蓝铜矿） | ★★★ |
| 磁铁矿 | 河床黑色重砂，用木棒悬挂测试磁性 | ★★☆ |
| 木材 | 遍地都是 | ★ |
| 植物纤维 | 蕨类茎秆、树皮纤维 | ★ |
| 石制工具 | 燧石/黑曜石打制 | ★★ |
| 粘土 | 河岸粘土层 | ★ |
| 树脂 | 针叶树，切口收集 | ★ |

**Day 3-5: 铜加工**

自然铜（如果能找到）可以直接锤打：

```
自然铜块
  ↓ 冷锻（石锤反复敲打）
铜片 (厚 ~1-2mm)
  ↓ 继续锤打 + 退火 (放入火中加热至暗红再缓冷，消除加工硬化)
铜带 (厚 ~0.3-0.5mm, 宽 ~3-5mm)
  ↓ 手工搓/滚成圆截面
粗铜线 (直径 ~1-2mm)
  ↓ 通过石头上钻的渐缩孔拉拔 (原始拉丝板)
铜线 (直径 ~0.5-1mm)
```

如果找不到自然铜，需要冶炼孔雀石（跳到 Era 2 的铜冶炼，但那需要更久）。

**临时替代**: 用捡到的自然铜锤成铜带（不拉丝），绕成扁线圈。效率低但可工作。

**Day 5-10: 建造发电机**

```
结构简图 (侧视):

        水流方向 →
          │
    ┌─────▼─────┐
    │   水轮     │ ← 木制水轮，直径 ~60cm
    │  ┌─────┐  │    叶片：劈开的木板
    │  │ 轴  │  │
    └──┤     ├──┘
       │     │
       │  ┌──▼──┐
       │  │磁铁 │ ← 磁铁矿块，固定在旋转轴上
       │  │矿块 │
       │  └──┬──┘
       │     │
       │  ┌──▼──┐
       │  │铜线 │ ← 铜线绕成的线圈（固定不动）
       │  │线圈 │    ~50-100 圈
       │  └─────┘
       │
       └──── 输出导线
```

**关键参数**：
- 磁铁矿的磁场强度很弱（~0.005T），远不如现代永磁体
- 需要尽可能多的磁铁矿和尽可能多的铜线圈匝数来补偿
- 预期输出：~1-5V AC，需要整流
- 整流方案：**机械整流器**（换向器）——旋转轴上的分瓣铜环 + 固定碳刷（木炭块）

**Day 10-15: 充电电路**

笔记本充电需要稳定直流电。方案：

1. 拆开笔记本，找到电池（通常是 3-4 串锂电芯，总压 11.1V 或 14.8V）
2. 发电机输出通过机械换向器变成脉动直流
3. 串联一个手工电阻（细长碳棒）限制电流，避免过充
4. **关键安全措施**：用 AI 教你的知识判断充电状态：
   - 电池发热 → 立即断开
   - 用 AI 帮你计算合适的限流电阻值
   - 每次充电时人工监控，不能无人值守

> **里程碑**: 笔记本重新亮起。从此你拥有了永续的 AI 顾问。

**Day 15-30: 改进发电系统**

- 增加水轮效率（建导水槽，提高水头）
- 增加铜线圈匝数
- 尝试制作多极发电机（多块磁铁矿交替排列）
- 目标：稳定输出 30-50W，足以正常使用笔记本

## Era 2: 冶金基础（Month 1 - Month 6）

笔记本已经有稳定电源了。现在开始真正的工业化进程。

### 2.1 木炭生产

所有冶金的第一步——**高温燃料**。

```
木炭窑截面图:

         覆土层 (粘土+泥)
    ╔═══════════════════╗
    ║                   ║
    ║    木材堆叠        ║  ← 干燥硬木，纵向竖放
    ║    (约 1m³)        ║
    ║                   ║
    ╚═══════╦═══════════╝
            ║ 进气口 (底部侧面，可封堵调节)
            ↓

温度: 300-500°C (缺氧热解)
时间: 2-3 天
产出: ~200kg 木 → ~50kg 木炭
```

白垩纪大气含氧 30%，火会烧得更旺——需要注意控制进气量，否则木炭会烧成灰。

### 2.2 铜冶炼（如果自然铜不够）

```
原料: 孔雀石 Cu₂(CO₃)(OH)₂ (绿色矿石)

          ┌─────────────┐
          │  粘土坩埚    │  ← 手工捏制，晾干后素烧硬化
          │             │
          │ 孔雀石碎块   │
          │ + 木炭层     │  ← 交替堆叠
          │ + 孔雀石     │
          │ + 木炭层     │
          └──────┬──────┘
                 │
          风箱鼓风 ← 皮囊风箱 (兽皮? 没有驯养动物...
                     用蕨叶编织的扇形手摇鼓风器)

温度: ~1100°C (铜熔点 1085°C)
产物: 底部聚集的铜锭 + 渣
```

**关键挑战**: 达到 1100°C。
- 普通篝火 ~600-800°C，不够
- 木炭 + 强制鼓风 → 可达 1200°C+
- 白垩纪高氧环境有利

**风箱替代方案**: 没有皮革（没有驯养的大型动物），可以用：
- 大型蕨叶扇手摇鼓风
- 黏土管制作的"吹管"（用嘴吹，多人轮换）
- 木制活塞风箱（用树脂密封）

### 2.3 铜线拉制

冶炼得到铜锭后，需要变成可绕线圈的铜线：

```
铜锭
  ↓ 热锻 (加热到暗红，锤打)
铜棒 (Ø~5mm)
  ↓ 退火 (加热后缓冷，软化)
  ↓ 通过硬石拉丝板 (渐缩孔序列)
  ↓   孔径: 4mm → 3mm → 2mm → 1.5mm → 1mm → 0.7mm → 0.5mm
  ↓   每 2-3 次拉拔后退火一次
铜线 (Ø0.5mm)
```

**拉丝板**: 用最硬的石头（石英、玛瑙）磨出渐缩锥形孔。
后期有了铁/钢之后，换成钢制拉丝板，可以拉到 Ø0.3mm。

**绝缘层**: 铜线绕制线圈需要匝间绝缘。方案：
- 松树脂 + 蕨类纤维缠绕（最容易获取）
- 漆：松树脂溶于松节油（松树脂干馏产物），涂覆后烘干
- 丝：蜘蛛丝（极细但量少）or 野蚕丝（白垩纪有蛾类）

### 2.4 青铜 (Month 2-3)

铜太软。铜 + 锡 = 青铜，硬度和耐磨性大幅提升。

```
锡石 (SnO₂, 棕黑色重砂) → 木炭还原 → 锡 (熔点 232°C，很容易)
铜 + 10%锡 → 青铜 (硬度翻倍，可铸造复杂形状)
```

青铜将用于：轴承套、齿轮、精密工具

### 2.5 铁 (Month 3-5)

```
块炼铁法 (Bloomery):

        烟囱 (粘土管)
          │
    ┌─────▼─────┐
    │  铁矿石    │  ← 赤铁矿 (Fe₂O₃, 红色) 或磁铁矿 (Fe₃O₄, 黑色)
    │  + 木炭    │     交替层叠
    │  + 铁矿    │
    │  + 木炭    │
    ├───────────┤
    │  风口      │ ← 鼓风
    └───────────┘

温度: ~1250°C
产物: 海绵铁 (bloom) — 多孔铁块，含渣
后处理: 反复加热锤打 (折叠锻造)，挤出渣滓
```

### 2.6 钢 (Month 5-6)

铁 + 控制碳含量 = 钢。

```
渗碳法:
  铁件 + 木炭粉 → 密封在粘土容器中 → 加热至 900°C 保持数小时
  → 碳渗入铁表面 → 表面变成钢 (含碳 0.3-0.8%)

淬火:
  加热至樱红色 (~800°C) → 快速投入水中
  → 马氏体转变 → 极硬

回火:
  淬火后加热至 200-300°C → 缓冷
  → 降低脆性，保留硬度
```

> **里程碑**: 你现在拥有了钢制工具。文明加速开始。

## Era 3: 精密制造（Month 6 - Month 18）

### 3.1 车床——精密制造之母

**这是整个科技树中最关键的单件装备。** 有了车床，才能制造圆柱形零件、轴承、螺纹。

```
第一代车床 (弓弦车床):

     弓 ──弦──→ 绕在工件上
                  ↓
    ┌──┐     ┌──────┐     ┌──┐
    │  │     │ 工件  │     │  │
    │支│     │      │     │支│
    │架│     │      │     │架│
    └──┘     └──────┘     └──┘
                ↑
              刀具 (钢制)

原理: 拉弓带动工件旋转，钢刀切削
局限: 只能单向间歇旋转，精度有限
```

```
第二代车床 (脚踏连杆车床):

              飞轮 (石制/铸铁)
               ↑
              曲柄
               ↑
            连杆
               ↑
            脚踏板

→ 连续旋转
→ 飞轮蓄能，转速更稳定
→ 精度: ±0.5mm (已接近 myCobot 水平!)
```

```
第三代车床 (水力驱动):

水轮 → 皮带传动 → 车床主轴
                 → 飞轮

→ 动力更大，可加工更硬的金属
→ 加上钢制导轨 + 进刀丝杠 → 精度 ±0.1-0.2mm
```

### 3.2 量具

精度需要量具来保证：

| 量具 | 制作方式 | 预期精度 |
|------|---------|---------|
| 直尺 | 钢板刻线，基于已知长度（笔记本尺寸）标定 | ±0.5mm |
| 卡钳 | 钢锻造，铰接式内/外卡钳 | ±0.3mm |
| 螺旋千分尺 | 精密螺纹 + 刻度套筒（需要先造出精密丝杠） | ±0.05mm |
| 角度规 | 钢板 + 刻度 | ±1° |
| 平板 | 三块铸铁板互研法（三板法确保绝对平面） | 平面度 0.02mm |

**三板互研法**值得特别说明——这是一个不需要任何参考标准就能产生绝对平面的自举方法：

```
取三块粗磨铸铁板 A, B, C

第一轮: A 与 B 互相研磨 → A,B 共形（但可能都是弧面）
第二轮: A 与 C 互相研磨 → A,C 共形
第三轮: B 与 C 互相研磨 → B,C 共形

因为三板必须两两共形，唯一的解就是三者都是平面。

重复多轮 → 精度逐渐收敛到 0.02mm 级平面度
```

### 3.3 螺纹

螺纹是精密机械的基石——丝杠、螺栓、调节机构全靠它。

```
第一根丝杠的自举:
1. 在车床上将钢棒车成精确圆柱
2. 用纸条以精确角度螺旋缠绕在圆柱上 (角度 = arctan(螺距/周长))
3. 沿纸条边缘用三角锉刻出螺旋槽
4. 初步螺纹完成 → 配合螺母（用同一丝杠翻模铸造）
5. 用这根丝杠做出带进刀丝杠的车床 → 车出更精密的丝杠
6. 重复迭代 → 精度收敛
```

### 3.4 齿轮

Arm A 需要减速齿轮。白垩纪版本不能做谐波减速器（需要特种钢薄壁柔轮），替代方案：

```
蜗轮蜗杆减速 (可达 50:1~100:1):
  蜗杆: 在车床上车螺纹 → 就是蜗杆
  蜗轮: 青铜铸造 → 车床上修形
  优点: 大减速比、自锁性
  精度: 回差可控制在 ~0.5°以内

摆线针轮减速 (可手工制造的高减速比方案):
  与谐波减速器类似的功能，但结构更粗犷
  用钢针 + 偏心轮实现
  减速比: 10:1~40:1
  需要精密车削但不需要特种材料
```

### 3.5 轴承

```
滑动轴承 (最先能做到):
  材料: 青铜套 + 动物脂润滑（白垩纪有小型哺乳类...或用植物油脂）
  精度: ±0.1mm (车削+研磨)

滚动轴承 (需要更高工艺):
  滚珠: 钢丝截断 → 两块淬火钢板间滚动研磨 → 球形
  套圈: 钢管车削 + 内圈沟道
  预期: 耗时数周手工研磨，但可行
```

## Era 4: 电气化（Month 12 - Month 30）

### 4.1 强力永磁体

Era 1 的磁铁矿太弱了。现在有了钢，可以制造真正的永磁体：

```
方法: 电磁感应磁化
1. 用铜线在铁芯上绕数百匝 → 电磁铁
2. 用水力发电机供电 → 强磁场
3. 将淬火钢棒放入线圈中 → 被磁化为永磁体
4. 淬火钢的矫顽力高 → 磁性持久

多次迭代:
  弱磁铁矿 → 第一代发电机 → 电磁铁 → 磁化钢棒
  → 更强的第二代发电机 → 更强的电磁铁 → 更强的永磁体
  → ... (正反馈循环)
```

### 4.2 电动机

有了强永磁体 + 大量绝缘铜线 + 钢芯，可以制造真正的直流电动机：

```
基础直流电机:
  定子: 永磁体 (2极或4极)
  转子: 铁芯 + 铜线绕组
  换向器: 分瓣铜环
  电刷: 碳棒 (木炭压制)
  轴承: 青铜滑动轴承

预期性能:
  功率: 5-20W (小型) 到 50-100W (中型)
  转速: 100-3000 RPM
  效率: 40-60% (手工制品，已经不错了)
```

### 4.3 步进电机（Arm A 的核心需求）

步进电机比直流电机更难做，但不是不可能：

```
简易两相步进电机:

  ┌──────────────────────┐
  │      定子齿 A         │  ← 铁芯 + 铜线绕组 A
  │   ┌──────────┐       │
  │   │ 转子齿   │       │  ← 齿形铁芯 (永磁或可变磁阻)
  │   └──────────┘       │
  │      定子齿 B         │  ← 铁芯 + 铜线绕组 B
  └──────────────────────┘

原理: 交替激励 A/B 绕组 → 转子步进
步距角: 取决于齿数 (12齿 = 30°/步, 24齿 = 15°/步)

手工极限: 24~48 齿 → 7.5°~3.75° 步距角
配合 50:1 蜗轮蜗杆 → 0.15°~0.075° 输出步距角
→ 在 350mm 臂展上 ≈ 0.9mm ~ 0.45mm 分辨率
```

**诚实评估**: 手工步进电机的步距角和平稳性远不如工业品。但配合编码器反馈（见 4.4）可以补偿。

### 4.4 位置反馈——替代编码器

没有半导体，无法制造光电编码器。替代方案：

```
方案 A: 机械式分度盘
  在输出轴上安装精密刻度盘 (车床刻线)
  用放大镜 (自磨玻璃透镜) 读数
  精度: ±0.1° (手工极限)
  问题: 无法电子读取，不能自动闭环

方案 B: 电位器式角度传感器
  滑动电阻 + 电刷
  电阻体: 碳粉 + 粘土烧制的薄片，或镍铬合金线 (需要镍...)
  替代: 细铜线密绕在绝缘管上 → 电阻随位置线性变化
  笔记本的音频输入口 (麦克风) 可以采集电压变化 → 算出角度
  精度: ±0.5° (受限于电刷接触噪声)

方案 C: 磁感应位置检测
  在输出轴上装永磁体
  固定一个铜线圈 → 旋转时感应出交流电压
  通过信号频率和幅度推算转速和位置
  需要笔记本做信号处理
```

**最优方案**: B + C 组合。电位器给粗定位，磁感应给转速。笔记本做闭环计算。

### 4.5 控制系统——笔记本作为大脑

没有微控制器，笔记本就是唯一的计算设备。

```
笔记本                    白垩纪制造的硬件
┌─────────────┐          ┌──────────────────┐
│             │          │                  │
│  音频输出    ├─────────→│ 继电器驱动电路    │
│  (3.5mm)    │ 模拟信号  │ (机械继电器      │
│             │          │  铁芯+铜线圈      │
│  音频输入    │←─────────┤  +弹簧触点)      │
│  (3.5mm)    │ 传感器信号│                  │
│             │          │ 4-8 路继电器      │
│  USB        ├─────────→│ → H桥 → 电机 ×6  │
│  (5V 电源)  │          │                  │
└─────────────┘          │ 位置传感器 ×6     │
                         │ → 音频输入        │
                         └──────────────────┘

控制原理:
  笔记本运行 Python 脚本
  通过音频输出发送不同频率的方波
  继电器响应特定频率 → 切换对应电机绕组
  位置传感器信号通过麦克风输入 → 笔记本采集分析
```

**频分复用**: 音频口只有 1 路输出，但可以同时叠加多个频率：
- 100Hz → 继电器 1（J1 电机 A 相）
- 200Hz → 继电器 2（J1 电机 B 相）
- 300Hz → 继电器 3（J2 电机 A 相）
- ...以此类推

每个继电器用 LC 谐振滤波器（铜线圈 + 自制电容器）只响应特定频率。

> 这整套系统本质上是 **用声音驱动机器人**。

### 与现代版 Arm A 的对照

| 组件 | 现代版 | 白垩纪版 | 精度影响 |
|------|--------|---------|---------|

| 末端执行器 | 真空吸嘴+电动夹爪 | 弹簧夹具 (手动/机械联动) | 简单但有效 |

### 预期精度

```
步进角: 7.5° (48齿步进电机)
蜗杆减速: 50:1
输出步距角: 0.15°
臂展: 350mm
末端分辨率: 350mm × sin(0.15°) ≈ 0.9mm / 步

加上电位器闭环: 可补偿到 ±0.5mm 左右
加上多次迭代校准: 可能接近 ±0.3mm
```

**结论: 白垩纪版 Arm A 精度约 ±0.3-0.5mm，与 myCobot 280 相当。**

不如现代版（±0.2mm），但考虑到这是从石头和泥土开始的，已经是奇迹了。
