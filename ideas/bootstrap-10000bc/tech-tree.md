# 科技树：从公元前一万年到机械臂

## Era 0: 黄金八小时（降落后 0-8h）

### 首要判断：你在哪？

打开笔记本，描述周围环境给 AI，判断你大致的位置。**选址至关重要**——直接决定了后续数月的效率。

**理想降落点排序**:

| 优先级 | 地区 | 为什么 |
|--------|------|--------|
| ★★★★★ | 安纳托利亚（今土耳其中部） | 铜矿丰富（后来的人类最早在此冶铜）、锡矿可达、气候温和、有人类部落 |
| ★★★★ | 新月沃地（今伊拉克/叙利亚北部） | 人口相对密集、野生小麦/大麦、靠近安纳托利亚铜矿 |
| ★★★ | 塞浦路斯附近海岸 | 岛上有大量高纯度铜矿（Copper 这个词就来自 Cyprus） |
| ★★★ | 多瑙河流域（今塞尔维亚） | 铜矿（Vinča文化后来在此最早冶铜之一）、铁矿、煤 |
| ★★ | 英格兰西南（康沃尔） | 全球最好的锡矿之一，但此时仍较冷 |

如果你不在上述位置，AI 会根据你描述的地形、植被、气候推算你的大致位置，并规划迁移路线。

> **如果运气好落在安纳托利亚**: 你脚下可能就踩着铜矿，步行范围内就有人类部落。这是开局天胡。

### 在断电前必须完成

与白垩纪版本相同的核心清单，但增加几项：

1. **知识转录**（同白垩纪版本）：矿石识别、冶炼炉图纸、发电机原理图
2. **语言学习策略**：记录基础沟通方法——
   - 指物命名法（指着东西学当地词汇）
   - 通用善意信号（分享食物、展示无害）
   - 用笔记本屏幕展示火焰/动物图片作为破冰手段
3. **地形评估 + 迁移规划**：如果不在矿产丰富区，规划迁移路线
4. **绘制"路线图海报"**：在一块大树皮上画出未来 3 年的科技树简图——这既是给自己的提醒，也是后来向部落伙伴解释"我们要做什么"的教具

### 关于"神迹"

> 一台会发光、能显示图像的平板物体，在公元前一万年的人类眼中，就是超自然现象。
>
> **善用但不滥用。** 用它来建立信任和权威，但不要吓到人。
> 理想操作：在安全距离内，展示屏幕上的火焰动画 → 好奇心 > 恐惧 → 接近 → 合作。

## Era 1: 建立根据地（Day 1 - Month 1）

### 白垩纪版本里最难的部分，在这里变得简单得多。

| 任务 | 白垩纪（独自一人） | 新石器（有人类同伴） |
|------|------------------|-------------------|
| 寻找铜矿 | 盲目搜索数天 | AI 指路 + 部落猎人带路 |
| 制作风箱 | 蕨叶手摇（效率极低） | **皮革风箱**（猎获兽皮） |
| 重体力劳动 | 独自锤打铜块 | 教会 2-3 人一起干 |
| 保护营地 | 独自面对猛兽 | 部落协防 |
| 食物/水 | 自行觅食 | 部落分享 |

### 1.1 与部落接触（Day 1-7）

**策略**：你不是征服者，你是"带来有用知识的奇怪旅人"。

```
Day 1-2: 观察
  - 寻找烟火/足迹/营地痕迹
  - 从远处观察部落规模和行为

Day 3-5: 接触
  - 携带"礼物": 用笔记本的手电筒功能在夜间发光 → 引起注意
  - 主动分享你猎获/采集的食物
  - 展示实用技能: 更高效的石器制作法（AI 教你的 Acheulean 技术）
  - 指物命名，开始学习他们的语言

Day 5-7: 融入
  - 帮助部落改进工具
  - 用 AI 知识识别药用植物，治疗小伤 → 建立信任
  - 开始教授"新技能": 编篮、简易陷阱、更好的火堆结构
```

**语言**: 公元前一万年的语言已经和现代语言完全不同，没有捷径。但人类语言学习能力是天生的——沉浸式学习，预计 2-4 周可达基础沟通，3 个月可流利对话。AI 可以帮你建立词汇表和语法推断。

### 1.2 电力求生（Day 3-15）

核心方案与白垩纪版本相同，但关键差异：

**皮革风箱带来的冶金优势提前了一步——但第一步仍然是为笔记本充电。**

充电路径不变：
1. 找到自然铜（地表露头/河床）→ 锤成铜线
2. 找到磁铁矿（黑色重砂，吸铁）
3. 建造水力发电机

**新增优势**:
- 可以请部落猎人帮你寻找"黑色吸石头的石头"和"红色软石头"——他们对本地矿产分布远比你熟悉
- 多人协作锤打铜线，速度快数倍
- 兽皮可以制作防水材料保护电气设备

**里程碑**: Day 10-15，笔记本重新充电成功。

### 1.3 组建"工程团队"（Day 7-30）

```
你需要从部落中招募和培养:

"铁匠学徒" ×2-3 人
  - 选标准: 对新事物好奇、体力好、手稳
  - 训练内容: 火候控制、锤打节奏、基础铸造
  - 他们将成为冶金生产的核心劳动力

"木工" ×1-2 人
  - 已有技能: 部落成员本来就会加工木材
  - 升级训练: 更精确的切割、卯榫结构
  - 负责: 水轮、车床框架、模具

"采矿/采集" ×2-3 人
  - 利用部落猎人的地形知识
  - 教授矿石识别方法
  - 建立固定采矿路线
```

> **关键心理学**: 你带来的每一个"新技术"都必须让他们看到直接好处。
> 不要说"我们要做一个可以自己动的手臂"——说"我教你做一把永远不会钝的刀"（钢刀 vs 石刀）。
> 先用冶金的直接成果（铜刀、铜针、铜鱼钩）证明价值，再推进更抽象的目标。

## Era 2: 冶金（Month 1 - Month 5）

> 核心流程与白垩纪版本相同（木炭→铜→青铜→铁→钢），但有人力和皮革风箱加持，速度快 2-3 倍。

### 与白垩纪版本的关键差异

| 环节 | 白垩纪 | 新石器 | 时间节省 |
|------|--------|--------|---------|
| 木炭生产 | 独自建窑、独自伐木 | 3-4 人协作，持续生产 | 2x |
| 鼓风 | 蕨叶手扇 / 吹管（肺活量极限） | **双囊皮革风箱** → 稳定 1200°C+ | 关键突破 |
| 铜冶炼 | 小坩埚，每次产铜 ~100g | 大型竖炉 + 持续鼓风，每次 ~500g-1kg | 5x |
| 铜线拉制 | 独自锤打，极度疲劳 | 一人加热退火，一人拉拔，一人收线 | 3x |
| 铁冶炼 | 独自操作 bloomery | 2人交替鼓风 + 1人添料 = 可持续运行 | 必要条件 |

### 2.1 皮革风箱（关键装备）

```
双囊风箱:

    ┌──────┐  ┌──────┐
    │ 皮囊A │  │ 皮囊B │  ← 鹿皮/牛皮缝制
    │      │  │      │     容积约 10L/个
    └──┬───┘  └──┬───┘
       │          │
       ▼          ▼
    ┌─────────────────┐
    │   Y形汇合管      │  ← 木管 + 粘土密封
    └────────┬────────┘
             │
             ▼
         风口 → 炉膛

操作: 交替按压两个皮囊 → 连续气流 → 稳定高温
一个人可以持续操作，不像吹管需要换气
```

这是公元前一万年版本最大的单一技术优势。白垩纪没有大型动物皮革，只能用植物材料做极低效的鼓风装置。

### 2.2 冶金时间线

```
Month 1:     木炭窑投产 + 第一炉铜（自然铜重熔或孔雀石还原）
Month 1.5:   铜工具产出: 铜刀、铜凿、铜锤、铜针
             → 部落生活质量飞跃 → 你的威望和团队规模上升
Month 2:     找到锡矿（锡石）→ 青铜合金
Month 2.5:   青铜工具: 更硬的刀、凿、斧 → 加工效率再升一级
Month 3:     开始铁冶炼（bloomery）
Month 4:     铁器产出 + 渗碳法制钢
Month 5:     钢制工具: 钢刀、钢锉、钢钻 → 可以加工其他金属了
```

### 2.3 铜线大规模生产

有了团队协作，铜线生产可以流水线化：

```
Station 1: 冶炼        (2人) → 铜锭
Station 2: 热锻拉棒    (2人) → Ø5mm 铜棒
Station 3: 退火+拉丝   (2人) → 逐级拉至 Ø0.5mm
Station 4: 绝缘涂覆    (1人) → 蜂蜡+树脂涂层 或 棉线缠绕

日产量估计: ~5-10m 绝缘铜线
一台小型电机需要: ~50-100m
→ 10-20 天可生产一台电机所需的全部铜线
```

**绝缘材料（比白垩纪大幅改善）**:

| 材料 | 来源 | 耐温 | 适用性 |
|------|------|------|--------|
| 蜂蜡 | 野蜂巢 | ~60°C | 低温绕组优选 |
| 松树脂漆 | 松脂+松节油 | ~100°C | 通用绝缘漆 |
| 棉纤维缠绕 | 野生棉（热带）| 高 | 层间绝缘最佳 |
| 丝线缠绕 | 野蚕 | 高 | 精细线圈 |
| 天然橡胶 | 热带橡胶树（如果在热带） | ~80°C | 柔性绝缘 |

## Era 4: 电气化（Month 10 - Month 20）

> 核心技术与白垩纪版本相同：
> - 电磁感应磁化 → 钢制永磁体
> - 铜线圈 + 永磁体 + 换向器 → 直流电动机
> - 笔记本音频口 → 频分复用 → 继电器 → H 桥 → 电机控制
>
> 详见 `../cretaceous-bootstrap/tech-tree.md` Era 4。

### 新石器版本的改进

**1. 更好的电机**

蜂蜡/棉纤维绝缘远优于纯松脂绝缘 → 线圈可以绕得更紧密 → 更强的磁场 → 电机功率密度更高。

**2. 更好的继电器**

皮革弹片 + 钢触点 → 更可靠的机械继电器。白垩纪版本只有木制弹片。

**3. 人力冗余**

调试电气系统时，可以让一个人负责转动手摇发电机测试，另一个人观察继电器动作，你自己操作笔记本。三个人比一个人效率高得多。

## Era 5: 新石器版 Arm A（Month 18 - Month 30）

### 组件清单对照

| 组件 | 现代版 | 新石器版 | 关键差异 |
|------|--------|---------|---------|
| J1-J3 电机 | NEMA17 闭环步进 | 手工步进电机 (24-48齿) | 步距角粗糙，靠编码器闭环补偿 |
| J1-J3 减速 | 谐波减速器 | 蜗轮蜗杆 50:1 (青铜蜗轮+钢蜗杆) | 回差更大，但功能等效 |
| J4-J6 电机 | NEMA14/11 步进 | 小型直流电机 + 蜗杆 | 更简单，精度靠位置反馈补偿 |
| 轴承 | 交叉滚子轴承 | 青铜滑动轴承 (动物脂润滑) | 摩擦更大但精度可接受 |
| 臂杆 | 碳纤维管 | 锻钢管 或 硬木 | 更重，刚性用钢管时可接受 |
| 编码器 | AS5600 磁编码器 | 铜线电位器 + 磁感应检测 | 分辨率低，笔记本做软件补偿 |
| 控制器 | STM32F407 | 笔记本音频IO + 继电器阵列 | 延迟大但能工作 |
| 末端 | 真空吸嘴 / 电动夹爪 | 弹簧夹具 (脚踏联动或小电机驱动) | 更粗犷 |
| 线缆 | 屏蔽铜线 | 蜂蜡绝缘铜线 + 棉线编织外皮 | 功能等效 |

### 预期精度

```
手工步进电机: 48齿 → 7.5°/步
蜗杆减速: 50:1
输出步距角: 0.15°
臂展 350mm → 末端分辨率 ~0.9mm/步

+ 电位器闭环 (精度 ±0.5°) → 有效分辨率 ~0.03° → ~0.18mm
+ 多次标定校准 → 实际重复精度: ±0.3-0.5mm
```

**与 myCobot 280 (±0.5mm) 基本持平。**

## 风险清单

| 风险 | 后果 | 应对 |
|------|------|------|
| 部落敌意 | 被驱逐或更糟 | 先展示善意和实用价值；避免展示"神迹"引发恐惧 |
| 笔记本在充电前损坏 | 失去 AI，但有手写笔记 | Era 0 知识转录是保险；之后也可凭记忆继续（速度大降） |
| 找不到铜矿 | 无法启动冶金 | AI 指导迁移到已知铜矿区；最坏情况先用陨铁 |
| 部落迁徙 | 失去团队 | 在矿区附近建立定居点 + 展示冶金成果让部落愿意留下 |
| 疾病 | 你对古代病原体可能缺乏免疫 | 保持饮水和食物卫生；AI 指导草药医学 |
| 铁器引发部落冲突 | 军事力量失衡 → 战争 | 有意识地控制武器扩散；优先制造工具而非武器 |
| 你的技术被视为巫术 | 社会性排斥 | 将技术包装为"技艺"而非"魔法"；让学徒亲手操作以去神秘化 |
