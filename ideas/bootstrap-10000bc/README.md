# 新石器自举计划

## 前提设定

你带着一台能"虚空连接" AI Agent 的笔记本电脑，穿越到了 **公元前 10,000 年**。

### 你降落的世界

末次冰期刚刚结束，全球正在升温。这是中石器时代向新石器时代过渡的前夜——人类已经存在了几十万年，但还没有发明农业、陶器或冶金术。

| 条件 | 状况 |
|------|------|
| 人口 | 全球约 500-1000 万人，分散的狩猎采集部落 |
| 技术水平 | 石器、骨器、兽皮帐篷、火的使用、弓箭 |
| 动物 | 狼（可驯化为狗）、野牛、野马、野羊、野猪、鹿 |
| 植物 | 野生小麦/大麦（新月沃地）、橡树、松树、桦树、各种果树 |
| 材料 | 燧石、黑曜石、粘土、天然铜（极稀少地表露头）、金砂 |
| 气候 | 温带区域宜居，北方仍有冰川退缩中 |
| 大气 | 与现代几乎相同 (21% O₂, ~280 ppm CO₂) |

### 你的压倒性优势

1. **AI** — 你随身携带了全人类的科技知识
2. **人类同伴** — 可以沟通（需要时间学语言）、教学、招募帮手
3. **可驯化动物** — 牛皮做风箱、马匹做动力、羊毛做绝缘
4. **已知的地理** — 你知道哪里有铜矿、锡矿、铁矿、煤矿
5. **完整的现代生态** — 蜂蜡、天然橡胶（热带）、棉花（野生）、蚕丝全部存在

### 最终目标

从纯石器时代出发，手工制造出 Arm A "巧手"精密机械臂。

## 与白垩纪版本的关键差异

> 白垩纪版本见 `../cretaceous-bootstrap/`（留作对照参考）

| 维度 | 白垩纪 (1亿年前) | 新石器 (公元前1万年) | 影响 |
|------|-----------------|-------------------|------|
| 人类 | 不存在 | 可招募帮手 | 劳动力 ×10+ |
| 皮革 | 不可用 | 猎获兽皮 | **风箱！** 冶金效率跃升 |
| 润滑 | 植物油脂（有限） | 动物脂肪（充足） | 轴承寿命提升 |
| 绝缘 | 松树脂（勉强） | 蜂蜡 + 天然橡胶 + 棉纤维 | 电线绝缘质量跃升 |
| 矿产定位 | 随机搜索 | AI 指路（已知矿床） | 数周 → 数天 |
| 体力劳动 | 独自一人 | 部落协作 | 冶炼/锻造产量大增 |
| 氧气 | 30%（火灾风险） | 21%（正常） | 需要更好的鼓风 |

**结论: 有人类同伴的版本，时间线从 3-5 年压缩到约 2-3 年。**

## 以下内容为Agent哥锐评环节

完成了。新版本在 ideas/bootstrap-10000bc/，白垩纪版本保留在 ideas/cretaceous-bootstrap/ 作为对照。
核心差异总结——公元前一万年版本的三大加速器：
皮革风箱 — 白垩纪版本最大的瓶颈是鼓风。没有皮革就没有风箱，没有风箱就烧不到 1200°C，冶金效率极低。有了兽皮，冶炼速度直接提升 5 倍。
人类同伴 — 从串行变成并行。你专注精密加工，学徒负责冶炼和锻造毛坯，木工维护水轮和车床。这是从"一个人的文明"变成"一个团队的工业革命"。
蜂蜡+棉纤维绝缘 — 电气化阶段的电机质量直接取决于绝缘材料。松脂勉强能用，但蜂蜡和棉纤维缠绕的线圈密度和可靠性远远更好。
最后那张"文明副产品"表我觉得很有意思——你一个人加一个部落用 2-3 年走完 12,000 年的科技路，本质上就是知识消除了试错。这其实也是 Embodied Agent 项目的核心逻辑。
